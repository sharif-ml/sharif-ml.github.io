The course is taught by Dr. Motahari who is an assistant professor at the department of computer engineering.
In this course, we will cover the following topics:

  • Introduction to Machine Learning and Statistical Learning
  • Statistical Learning
  • Classification
  • Resampling Methods
  • Linear Model Selection
  • Tree-based Methods
  • Support Vector Machines
  • Neural Networks and Deep Learning
  • Unsupervised Learning
Note: In this semester, we will put a huge focus on neural networks and deep learning. More precisely, you'll learn about the following concepts:

  • Neurons and Perceptrons
  • Gradient Descent
  • Backpropagation
  • Feed-Forward Neural Networks
  • Recurrent Neural Networks (RNNs) including Long Short-Term Memory (LSTM) and Gated Recurrent Unit (GRU)
  • Convolutional Neural Network (CNN)
  • AutoEncoders - VAEs
  • Generative Adversarial Networks (hopefully we'll have time for this!)
We have considered the following books as our main references:
  • James, G., Witten, D., Hastie, T., Tibshirani, R., & Taylor, J. (2023). An Introduction to Statistical Learning: With Applications in Python. New York: springer.
  • Hastie, T., Tibshirani, R., Friedman, J. H., & Friedman, J. H. (2009). The elements of statistical learning: data mining, inference, and prediction (Vol. 2, pp. 1-758). New York: springer.
